Programming Experience

Introduction

I was first introduced to programming through MIT's UI-based programming platform Scratch. In high school I began to learn more about "real" programming languages such as C and Java. Finally, in university, I greatly expanded my coding horizons.

Other languages and libraries I have used include:

  • Java
  • Python
  • C#, .NET, Unity
  • HTML and JavaScript
  • C++
  • SQL and MongoDB

One huge idea in software development is that all OOP languages have remarkably functionality. The important concepts (loops, variables, objects, etc.) can be applied everywhere. With a syntax sheet and some decent documentation, I have no doubt that I could learn any language that the situation requires.

Forge and Java

Forge API is a code library designed to allow modifications to the open-source game "Minecraft." The game and API are written in Java. One summer long ago, I jumped in head-first by teaching myself Java in order to learn to use the Forge API.

My use of Java improves continously as I learn new skills and best practices. Also, working with other modders over the years has given me a chance to learn many skills, such as how to work with a development team and how to communicate properly.

A YouTuber generously showcased my first mod, released in 2015. Their video can be found here.

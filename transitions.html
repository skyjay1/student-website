Blog Post: CSS Transitions

What are CSS Transitions?

According to w3schools, CSS transitions allows you to change property values smoothly, over a given duration. [1] This can be used to create smooth and simple animations, change color and size, and perform other transformations. CSS Transitions are not supported by Internet Explorer 9 or earlier versions.

To see the power of CSS Transitions, hover your mouse over the following figure. No Javascript or plugins are used - just pure CSS.

How it works

CSS Transitions specify an initial state and a final state, and intermediate states are automatically calculated [2]. They are easily used in conjunction with psuedoselectors such as :hover.

How to use

CSS Transitions require a minimum of two things:

  • the transition-property to modify
  • a transition-duration over which the transition will take place

If the duration is not specified, it will default to 0 and the entire purpose of the transition is lost.

For example, the following code will make a red box that, when hovered over by the mouse, will transition the width property from 50px to 200px over the duration of 2 seconds:

div {
  width: 50px;
  height: 100px;
  background-color: red;
  transition-property: width;
  transition-duration: 2s;
}

div:hover {
  width: 200px;
}

Customizing transition properties

Maybe we want something slightly different. One tweak we can make is by changing the duration from 2s to 4s as below:

But what if this isn't quite what we wanted? There are two other CSS properties are used to customize the transition style:

  • transition-delay
  • transition-timing-function

We can change each of these properties to adjust the original example.

transition-delay

Let's add transition-delay: 1s

This will, as the name suggests, create a delay before the transition is played. In our case, the transition waits until the mouse has been hovering for 1 second before playing.

Notice that the transition is affected both ways -- there is a delay before beginning to grow and, once the mouse is removed, there is a delay before it returns to its original size. This symmetry applies to all of the transition properties.

transition-timing-function

Let's adjust the timing function. This specifies the speed curve of the transition - right now it defaults to ease, but there are other options. Here is a full list:

  • linear - same speed from start to end
  • ease-in - starts slow, then fast
  • ease-out - starts fast, then ends slowly
  • ease-in-out - starts slow, then fast, then ends slowly
  • ease - similar to ease-in-out but starts slightly faster than it ends
  • cubic-bezier(n,n,n,n) - takes cubic-bezier parameters to calculate a custom ease in and ease out function

Here's a look at each of these in action. Notice that all of the transitions take 2 seconds, but the rate of change is different for each one.

Using the transition property

The transition property is a nice shorthand that combines several of the transition properties we've discussed. It is often preferred for its ability to specify all of the transition properties at the same time.

The following are equivalent:

transition-property: width;
transition-duration: 2s;
transition-timing-function: linear;
transition-delay: 1s;
transition: width 2s linear 1s;

Besides combining the property and duration values, the transition property accepts a comma-separated list to enable multiple transitions at the same time. For example, if we wanted our box to change color as well as grow, we can do the following:

div {
  width: 50px;
  height: 100px;
  background-color: red;
  transition: width 2s, background-color 2s;
}

div:hover {
  width: 200px;
  background-color: yellow;
}

Note that we are using the transition property as a shorthand for only two items: transition-property and transition-duration. The order is, of course, very important.

If each of the desired transitions will take the same amount of time, the all value can be used instead. This will automatically detect transitionable changes between the two states and transition each one at the same time.

Example usage:

transition: all 2s;

Other transitionable properties

As demonstrated in the earlier example, transitions can be used on much more than just the width of an element. The following can be especially useful for creative and neat-looking transitions:

  • background-color
  • scale
  • opacity
  • transform
  • color
  • flex-grow (if using flexbox)
  • font-weight

A full list of properties that can be animated (according to MDN) can be found here.

Summary

CSS Transitions gives programmer powerful and well-supported tools for simple, clean, neat-looking animations.

Every transition has the following properties:

  • transition-property (required)
  • transition-duration (required, defaults to 0s)
  • transition-delay (defaults to 0s)
  • transition-timing-function (defaults to ease)

The most commonly-used property is transition, which has the power to specify multiple transitions at once.

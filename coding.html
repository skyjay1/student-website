ActionScript

ActionScript is one of the programming languages supported by Adobe Flash. The main lesson I learned from working in ActionScript is that most OOP languages are essentially the same. The only two things I need to work in any unfamiliar language are:

  1. A basic knowledge of core features (loops, variables, objects, etc.)
  2. Access to a language-specific syntax sheet

With these two, I was able to build fun animations and small games using Adobe Flash. And I learned many valuable lessons about programming and animation along the way.

Forge and Java

Forge API is a code library designed to allow modifications to the open-source game "Minecraft." The game and API are written in Java. One summer long ago, I jumped in head-first by teaching myself Java in order to learn to use the Forge API.

My use of Java improves continously as I learn new skills and best practices. Also, working with other modders over the years has given me a chance to learn many skills, such as how to work with a development team and how to communicate properly.
